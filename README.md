## 关于本程序

本程序适用于各种同城社区交友类产品，例如同城相亲、同城找搭子、**仿小红书生活笔记**、同城活动动态等等。

界面美观功能齐全，代码简洁优雅。**带后台管理和网页端**。

兼容安卓端，H5网页，微信小程序。

## 本程序使用的技术

- 手机端 小程序：uniapp
- 网页端：vue
- 后端：python fastapi
- 后台：vue
